ごちゃついててごめんなさい。magic.pyとmissions.pyが魔法陣ゲームのルールです。dqn_1_CNN2.ipynbとdqn_1_CNN3.ipynbが強化学習のコードです。学習結果のパラメータがskptに入っています。
dqn_1_CNN2.ipynbはミッション0とミッション1との対戦を学習させています。agent_CNN2_0_qnet.ptはミッション0側の、agent_CNN2_1_qnet.ptはミッション1側の学習済みパラメータです。人間よりかは弱いです。
dqn_1_CNN3.ipybnもほぼ同様ですが、報酬（次の手が勝利手か敗北手かそれ以外か）は学習関係なく与えられるようにしてあります。勝利手の見逃しや自爆がないということです。
